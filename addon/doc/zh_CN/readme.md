# 大写字母选择提示

一个为 NVDA 实现文本选择时大写字母提示功能的原型插件。

## 目的

本插件是以下功能请求的**概念验证实现**：
- [Issue #4874](https://github.com/nvaccess/nvda/issues/4874)
- [Issue #12996](https://github.com/nvaccess/nvda/issues/12996)

## 功能

当选择单个字符时（例如使用 `Shift+光标键`），NVDA 将使用与逐字符导航相同的方式提示大写字母：

- **大写字母提示音** - 在大写字母前发出短促蜂鸣声
- **朗读"大写"** - 在大写字母前朗读"大写"一词
- **大写字母音调变化** - 朗读大写字母时使用更高的音调

## 配置

**本插件没有任何设置。** 它遵循您现有的 NVDA 语音设置：

1. 打开 NVDA 菜单 → 选项 → 设置 → 语音
2. 配置遇到大写字母时的朗读选项：
   - "大写时音高变化的百分比"
   - "遇到大写字母时先提示大写"
   - "遇到大写字母时发出提示音"

插件会自动遵循这些设置。
